4 Weeks of Work

From basic responsive layouts to a complete mini e-commerce capstone website.

Week 1

Responsive Landing Page & Countries App

In Week 1, I completed Task 1: Responsive Landing Page and Task 2: Dynamic Content with JavaScript.

  • Built a clean responsive landing page for a fictional gaming service.
  • Added sticky navbar, hero section, CTA button, feature cards, and footer.
  • Used CSS media queries, Flexbox, and Grid for responsive design.
  • Created a JavaScript app using the REST Countries API.
  • Displayed country flags, population, live search, loading spinner, and API error handling.

This week improved my understanding of responsive web design, DOM manipulation, Fetch API, loading states, and building clean UI without frameworks.

Week 2

JavaScript Quiz App & Multi-Step Form

Week 2 focused on building interactive web projects using HTML, CSS, and JavaScript.

  • Created a quiz app with 10 questions loaded from a JavaScript array.
  • Displayed one question at a time with 4 answer options.
  • Highlighted correct and wrong answers instantly.
  • Added score tracking, progress display, 30-second timer, and restart option.
  • Built a 3-step registration form with validation and localStorage.
  • Added saved entries table, delete option, and password strength indicator.

This week helped me improve DOM manipulation, JavaScript arrays, timers, validation logic, localStorage, responsive design, and user interaction.

Week 3

Mini E-Commerce Page & CSS Framework Project

In Week 3, I worked on two practical frontend development tasks that improved my skills in JavaScript, localStorage, responsive design, and CSS frameworks.

  • Built a realistic mini e-commerce product page.
  • Added product listing page and product detail page.
  • Used 12+ products from JavaScript data.
  • Added category filter, price sorting, and add to cart functionality.
  • Added quantity update, remove item option, cart total, and item count update.
  • Saved cart data using localStorage.
  • Rebuilt a previous landing page using a CSS framework.
  • Used navbar, cards, buttons, modal popup, forms, and responsive grid layout.

This week helped me understand how real shopping cart logic works and how CSS frameworks make development faster and cleaner.

Week 4

Portfolio Write-up & Demo

In Week 4, I completed the final wrap-up task for my capstone website by preparing a professional portfolio write-up and demo plan.

  • Documented the capstone website as a professional portfolio piece.
  • Wrote the project write-up with features, tech stack, challenges, and solutions.
  • Wrote a reflection on my journey from plain HTML/CSS to a full e-commerce site.
  • Listed future technologies to learn, including React and Node.js.
  • Added all 4 weeks of work into this simple portfolio page.
  • Included live project URL, portfolio link, LinkedIn, and GitHub links.

This final task helped me present my work professionally and reflect on my progress throughout the internship.
